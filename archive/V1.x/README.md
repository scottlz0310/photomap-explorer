## アーカイブ理由

これらのファイルは以下の理由でアーカイブされました：

1. **main.py から直接参照されていない**
   - 現在のエントリーポイント（`main.py` → `FunctionalNewMainWindow`）から呼び出されていない

2. **vulture による未使用コード検出**
   - 静的解析により、未使用の関数・クラス・変数として検出された

3. **新アーキテクチャでの置き換え完了**
   - v2.1.2の新UIアーキテクチャで機能的に置き換えられた

## 復活の可能性
